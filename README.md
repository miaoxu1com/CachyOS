# CachyOS

## 源切换工具
### ArachLinux源配置
```shell
kate /etc/pacman.conf
[archlinuxcn]
Server = https://mirrors.aliyun.com/archlinuxcn/$arch
```
### paru安装

```shell
pacman -Sy paru
```
### chsrc安装

```shell
paru -S chsrc-bin
```

### 切换源

```shell
chsrc set python
chsrc set node
chsrc set rust
chsrc set java
```
